# Skill 01 — Repo Genesis

Purpose: create the initial Rust/Cargo `comptext` CLI scaffold.

Allowed:
- README.md
- LICENSE
- Cargo.toml
- comptext.example.toml
- src/main.rs
- src/cli.rs
- docs/**
- .agent/skills/**
- .agents/skills/**
- .github/workflows/ci.yml
- tests/**

Forbidden:
- provider calls
- network calls
- reading API keys
- git commit/push

Required commands:
- `ctxt --help`
- `ctxt doctor`
- `ctxt providers list`
- `ctxt version`

## Minimal MCP stack

- Git MCP: read repo topology, history, diffs.
- Developer Knowledge / Docs MCP: fact-check APIs and SDKs.
- Internal Docs/RAG MCP: optional later.
- CI/Logs MCP: optional later and read-only.

## Excluded from minimal stack

- database write MCP
- cloud deploy MCP
- secrets manager with secret output
- PM/email/calendar tooling for core build
